# REFORMATOR BIO

Premium Health Intelligence System — интеллектуальная система управления здоровьем, анализами и мужским гормональным балансом.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Run locally

```sh
npm i
npm run dev
```

## Deploy

Build and deploy (e.g. Vercel):

```sh
npm run build
```
